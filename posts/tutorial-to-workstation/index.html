Tutorial to workstation

2020-12-18 02:46
  • SSH 隧道
    1. 内网穿透
      1. ssh 内网穿透
      2. autossh （ssh 增强）

    2. 登录

      ssh user@hostname

    3. 无密码登录

      需要上传本地密钥

      1. 生成本地密钥 ssh-keygen
      2. 上传本地密钥 ssh-copy-id
    4. 更快登录
      1. 设置 .ssh/config
  • 文件传输

    1. 采用 Filezilla 传输文件

      最方便的一种方式，根据 ssh 信息来配置即可

    2. 采用 scp 传输文件

      在实现 ssh 免密码的基础上， scp [filename] [remote-host] 即可

    3. 采用 git 同步文件

      需要 git 操作

  • 运行代码

    1. python 运行脚本

      python test.py

    2. 后台运行

      后台运行在命令后面加入 & ，也即命令成为 python test.py & 即可。

    3. 存储日志

      在代码中，在关键信息的地方打印日志即可。

    4. 存储图像

      代码后台中，无需使用 matplotlib.show 来查看图像，因此保存即可。

  • 打印关键信息和保存日志以及查看日志
    1. 打印日志
      1. 代码中添加需要打印的信息
    2. 保存日志

      1. 使用文件重定向

        例如 python test.py > log.txt

      2. 同时将错误信息重定向

        2>&1

    3. 查看日志
      1. 使用 Pycharm 设置 sftp 来查看文件，最方便
      2. ssh 登录远程机器，然后使用 more or less 命令查看
      3. Emacs 的 tramp, 次方便。
